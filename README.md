# ECU Flash Tool (Python)

## Overview

This project is a Python-based ECU Flash Tool developed during my software internship.

The tool provides a graphical interface to flash firmware files into an ECU (Electronic Control Unit) and manage firmware records using a database.

This project demonstrates backend development, GUI programming, and file handling.

---

## Features

✔ Firmware Flashing Tool  
✔ GUI Interface  
✔ Database Storage  
✔ File Selection  
✔ Flash Process Control  

---

## Technologies Used

- Python
- Tkinter (GUI)
- SQLite Database

---

## Project Files

create_db.py  
Creates the database for storing firmware details.

flash_gui.py  
Graphical User Interface for the flash tool.

flash_process.py  
Handles firmware flashing operations.
